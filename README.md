# Nessy Code Season

## Локальный запуск

Установить зависимости:

```bash
npm install
```

Обычный режим игрока:

```bash
npm run dev
```

Режим для отладки прохождения игр:

```bash
npm run dev:controls
```

В этом режиме доступны служебные кнопки:

- `Попробовать еще раз` — перезапустить текущую мини-игру.
- `Пропустить игру` — пропустить текущую мини-игру с результатом `fail`.

Можно включить только одну служебную кнопку:

```bash
npm run dev:skip
npm run dev:retry
```

При смене режима останови текущий dev-сервер и запусти нужную команду заново.
Все dev-команды используют `http://127.0.0.1:5173` и `--strictPort`.
Если порт уже занят, Vite завершится с ошибкой вместо тихого запуска на
следующем порту.
